## Deutsch (ᴅᴇ)

**MaikelsTPA** ist ein Minecraft-Plugin, das ein einfaches TPA System bereitstellt. Spieler können sich gegenseitig Teleport-Anfragen senden und annehmen oder ablehnen. Das Plugin ist kompatibel mit Spigot- und Paper-Servern.

### Features
- TPA-Anfragen an andere Spieler senden
- TPA-Anfragen annehmen oder ablehnen
- Chat Buttons zum annehmen oder ablehnen
- Zentraler TPA-Manager für die Verwaltung von Anfragen
- Reload-Command zur Aktualisierung der Config.yml (OP)
- Kompatibel mit Java 21

### Befehle
| Befehl        | Beschreibung                              |
|---------------|------------------------------------------|
| `/tpa <Spieler>` | Sendet eine TPA-Anfrage an einen Spieler |
| `/tpaccept`      | Akzeptiert die aktuelle TPA-Anfrage      |
| `/tpdeny`        | Lehnt die aktuelle TPA-Anfrage ab        |
| `/tpareload`     | Lädt die Plugin-Konfiguration neu        |

### Permissions
| Permission             | Beschreibung                        |
|------------------------|------------------------------------|
| `maikelstpa.tpa`       | Erlaubt das Senden von TPA-Anfragen |
| `maikelstpa.tpaccept`  | Erlaubt das Akzeptieren von TPA-Anfragen |
| `maikelstpa.tpdeny`    | Erlaubt das Ablehnen von TPA-Anfragen |
| `maikelstpa.tpareload` | Erlaubt das Ausführen des Reload-Commands |

### Installation
1. Lade die neueste Version von `MaikelsTPA.jar` herunter.  
2. Lege die `.jar` Datei in den `plugins` Ordner deines Servers.  
3. Starte den Server neu.  
4. Passe die Einstellungen in `config.yml` nach Bedarf an.

### Konfiguration
- **config.yml** enthält alle einstellbaren Optionen für das Plugin (z.B. Nachrichten).  
- Konfiguriere die Permissions über Luckperms

## English (ᴇɴ)

**MaikelsTPA** is a Minecraft plugin that provides a simple TPA system. Players can send and accept or deny teleport requests to each other. The plugin is compatible with Spigot and Paper servers.

```INFO: MaikelsTPA, like all other plugins from MaikelCollin, is configured in German by default, but this can be changed via the config.yml file.```

### Features
- Send TPA requests to other players
- Accept or deny TPA requests
- Chat buttons to accept or decline
- Central TPA manager for tracking requests
- Reload command to update the Config.yml (OP)
- Compatible with Java 21

### Commands
| Command         | Description                             |
|-----------------|-----------------------------------------|
| `/tpa <player>` | Sends a TPA request to a player         |
| `/tpaccept`     | Accepts the current TPA request         |
| `/tpdeny`       | Denies the current TPA request          |
| `/tpareload`    | Reloads the plugin configuration        |

### Permissions
| Permission             | Description                        |
|------------------------|------------------------------------|
| `maikelstpa.tpa`       | Allows sending TPA requests        |
| `maikelstpa.tpaccept`  | Allows accepting TPA requests      |
| `maikelstpa.tpdeny`    | Allows denying TPA requests        |
| `maikelstpa.tpareload` | Allows executing the reload command |

### Installation
1. Download the latest `MaikelsTPA.jar`.  
2. Place the `.jar` file into your server's `plugins` folder.  
3. Restart the server.  
4. Configure the plugin using `config.yml` as needed.

### Configuration
- **config.yml** contains all adjustable options for the plugin (e.g., messages).  
- Configure the permissions via **Luckperms**
